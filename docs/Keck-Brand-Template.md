# Keck Medicine of USC — Brand & Design Guidelines

## Overview

Use these guidelines whenever creating visual content for Keck Medicine of USC.
This covers slides, dashboards, documents, web prototypes, and any deliverable
that should carry Keck branding.

## Colors

### Primary Palette

| Name | Hex | Usage |
|------|-----|-------|
| Keck Red | #9B0000 | Primary brand color. Headers, title bars, accent borders, primary metric highlights. |
| Dark Red | #7A0000 | Hover states, secondary headers, background overlays. |
| Red Alert | #C0392B | Off-target warnings, at-risk indicators. Use sparingly. |

### Accent Colors

| Name | Hex | Usage |
|------|-----|-------|
| Gold | #F5C518 | Secondary accent. Section headers, speed/time metrics, in-progress indicators. |
| Dark Gold | #B8860B | Metric card accents, chart elements, time-based KPIs. |
| Olive Green | #7A8B2E | Pipeline/volume metrics, next-steps headers. |

## Typography

| Element | Font | Fallback |
|---------|------|----------|
| Headlines / Titles | Georgia | serif |
| Body Text / Labels | Calibri | sans-serif |

### Dashboard Sizes

| Element | Size |
|---------|------|
| Dashboard title | 24px bold Georgia |
| Big metric numbers | 28-36px bold Georgia |
| Metric card labels | 11px uppercase Calibri, 600 weight |
| Section headers | 15px bold Georgia |
| Body text | 11-12px Calibri |
| Source lines | 9px Calibri #AAAAAA |

### Slide Sizes

| Element | Size |
|---------|------|
| Slide title | 20-24pt bold Georgia |
| Section banner text | 12pt bold Calibri, white on color |
| Bullet text | 10pt Calibri |
| Source/footnote | 7-8pt Calibri, light gray |

## Design Rules

- Card borders: 1px solid #E8E6E1, 8px border-radius
- Card top accents: 3px colored top border for category
- Status dots: 10px circles (green/amber/red)
- Section banners: full-width colored rectangles, white or dark text centered
- Gold accent dividers: 3-4px horizontal lines, #F5C518
- Off-white (#F7F6F2) page background, white cards
- Trend arrows: ↑ green improving, ↓ red worsening
- Data source attribution (9px gray) at bottom of every panel
- Footer: navy bar with copyright left, session number right
